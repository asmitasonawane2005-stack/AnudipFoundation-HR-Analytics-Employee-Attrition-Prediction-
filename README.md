# 📊 HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an **HR Analytics Dashboard** created using Power BI to analyze employee data and generate insights for better decision-making.

---

## 📂 Dataset

The dataset contains HR-related information such as:

* Employee ID
* Department
* Salary
* Attrition
* Job Role
* Age
* Experience

📁 File: `dataset/HR_Dataset.xlsx`

---

## 📊 Dashboard Features

The Power BI dashboard includes:

* 🔹 Employee Count Analysis
* 🔹 Attrition Rate
* 🔹 Department-wise Distribution
* 🔹 Salary Insights
* 🔹 Age Group Analysis
* 🔹 Job Role Breakdown



## 🎯 Key Insights

* High attrition observed in specific departments
* Salary distribution varies across job roles
* Majority employees fall in mid-age group

---
